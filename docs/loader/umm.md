# UnityModManager Installation Guide

## Prerequisites

- [UnityModManager](https://www.nexusmods.com/site/mods/21) installed
- ADOFAI game

> [!WARNING]
> For ADOFAI 2.10.0 and above, use **UnityModManager 0.32.5.0** or higher, otherwise crashes may occur.

## Installation

1. Go to [Releases](https://github.com/Xbodwf/Iridium/releases) and download the Iridium build matching your ADOFAI version.

> Each release provides multiple builds for different ADOFAI versions — choose the one that matches your game version.

2. Extract the downloaded files to: `A Dance of Fire and Ice/Mods/Iridium`. Create the folder if it does not exist.

3. Launch the game. If the game is already running, restart it.
