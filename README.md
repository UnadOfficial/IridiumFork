# Iridium

An optimized mod for A Dance of Fire and Ice.

## Author
Xbodwf

## Project Structure
- `Iridium.csproj`: Project file
- `Main.cs`: Main mod entry point
- `Settings.cs`: Mod settings
- `Patches.cs`: Harmony patches
- `Info.json`: UMM mod info
- `Resources/`: Mod resources folder

## Setup
1. Copy required game DLLs to the `lib/` folder.
2. Open `Iridium.sln` in Visual Studio or Rider.
3. Build the project.
